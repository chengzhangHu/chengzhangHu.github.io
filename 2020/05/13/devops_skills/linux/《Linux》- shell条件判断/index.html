字符串非空

  • n 测试是否为非空，需要””避免异常错误

整数比较

  • -gt

  • -eq

  • -ne

  • -lt

  • -le

  • -ge (大于或等于)

文件属性判断

  • -e 文件或者目录是否存在

  • -d 判断存在且为目录

  • -f

  • -L 软连接 （类似快捷键，跨分区，但是删除不可用；硬链接，相反；）

  • -p 管道

  • -r 读权限

  • -w 写权限

  • -x 执行权限;root 也必须有x，才能执行

  • -s 非空
