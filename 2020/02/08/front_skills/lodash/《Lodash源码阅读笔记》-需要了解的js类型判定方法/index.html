前端开发，js是一个避开不了的一门技术。js的诞生为了美化交互。

js类型

  • null
  • undefined
  • boolean
  • string
  • number
  • Object
  • 引用类型(object、array、function)
  • symbol

js类型判定

null

  • 值确定，直接比较
  • typeof不能作为唯一判断
  • toString

undefined

  • 值确定，直接比较
  • typeof 不能作为唯一标准
  • toString

boolean

  • 值确定直接比较

  • typeof 写法

  • toString

  • lodash写法

string

  • typeof

  • toString

  • lodash写法

number

  • typeof

  • toString

  • lodash

Object

  • typeof

  • toString

  • lodash

Array

  • typeof 不能用
  • toString
  • Constructor
  • lodash

Function

  • typeof
  • toString
  • lodash

Symbol

  • typeof
  • toString
  • lodash
