自动部署方案

GitHub action

  • action 拉取失败

  • 原因：
    利用github action构建，构建的public发布到xxx.github.io路径。拉取部署action失败，导致自动部署失败！

  • 解决方案
    fork action到自己repository下，自己修改构建sh脚本。
