4. dev-tool – Augury

介绍

Augury是最常用的开发人员工具扩展，用于调试和分析Google Chrome和Mozilla Firefox浏览器中的Angular应用程序。

功能

Augury通过组件树和可视化调试工具帮助Angular开发人员可视化应用程序。开发人员可以立即洞悉其应用程序结构，变更检测和性能特征。
