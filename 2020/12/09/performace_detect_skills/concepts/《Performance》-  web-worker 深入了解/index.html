《Performance》- web-worker 深入了解

2020-12-09

web-worker

通信过程

web api支持情况

不支持的web api

  • window
  • document

支持的

  • navigator

  • Console API

  • CustomEvent

  • Promise

  • websocket

  • ajax XMLHttpRequest

  • location(read-only)

  • setTimeout、clearTimeout、setInterval、clearInterval

  • atob()、btoa()

  • Cache对象(IE不支持)

  • IndexedDB

webworker 会内存泄漏吗？

worker是new创建的，在angular destory中，webworker依旧处于监听状态，如果不销毁，会存在内存泄漏

postMessage可以传递哪些参数？

  • 字符串
  • 对象，但是对象方法会移除，
  • 数组
  • 函数，直接异常

如何终止webWorker

在angular开发中，我们需要手动关闭worker
