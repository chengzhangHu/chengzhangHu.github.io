《Linux》- windows安装虚拟机并配置ssh

2020-08-28

虚拟机介绍

VMware workstation Pro

场景：
让Windows用户使用Linux系统

支持度

  • 大多数64位Linux发行版
  • Win7+ （64bit）
  • Windows Server2008 R2+

优缺点

优点 缺点
不错的网络性能、可后台运行、先进3D解决方案 偶尔的错误、崩溃
支持定制，必然收费 尽管可以定制，但是选项较少

VMware Fusion Pro

场景：
让Macos用户使用Windows系统

支持度

  • macos/os X 10.9+

优缺点

优点 缺点
完美运行 Windows软件，且速度快 首次启动慢，存在偶尔兼容性问题
创建、恢复快照简单实用 非技术领域用户来说，关键解决方案不丰富

Parallels Desktop

macos/os X 10.9+

Oracle VM VirtualBox

支持度

  • 大多数64位Linux发行版
  • macOS/OS 10.9+
  • Solaris 10+
  • Win7+ （64bit）
  • Windows Server2008 R2+

优缺点

优点 缺点
免费开源 缺乏竞争对手功能升级、操作体验改进慢
跨平台、兼容性好、定制选项多 内存使用率较高、全屏模式可能无法渲染虚拟机

QEMU

场景
大多数用户将linux机器作为其主机

支持度

  • 大多数64位Linux发行版
  • macOS
  • Windows(32 or 64)

优缺点

优点 缺点
免费开源、免费且强大的虚拟化技术 配置麻烦
