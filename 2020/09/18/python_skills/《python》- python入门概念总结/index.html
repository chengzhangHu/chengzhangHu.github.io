数据类型

6种标准对象类型：

  • Number（数字）
  • String（字符串）
  • List（列表）
  • Tuple（元组）
  • Sets（集合）
  • Dictionary（字典）

3种数值类型

  • int 整型 - 正、负整数，不带小数点

    Python3.7中，整除，需要使用//， /得到的是浮点数

  • float 浮点型 - 整数部分与小数部
  • complex 复数 - 实数部分和虚数部分

    a+bj或complex(a,b) 表示

  • 常量： PI

变量

没有显示的var这类的声明，第一次默认就是变量，使用前，必须先赋值。
可以type(), 类似js typeof 判定类型
